bool leapYear ( int  year)

判定给定的年份是否是润年

参数
year年份。
返回
真：润年；假：不是润年。

非整百年份能被4整除，或整百年份能被400整除时，则该年份是润年，否则，不是润年。

判定方法：

  1. ((year % 100 != 0) && (year % 4 == 0)) || (year % 400 == 0)
  2. (year % (( year % 100 != 0) ? 4 : 400)) != 0
  3. !(year % ((year % 100) ? 4 : 400))，结合前一方法获知为何求反的原因（非0为真）
void MonthDay ( int  year,
int  yearDay,
int *  pMonth,
int *  pDay 
)
参数
year一个表示年份的整数
yearDay一个表示这一年第几天的整数
pMonth一个存储月份的整型变量的地址（出口参数）
pDay一个存储日的整型变量的地址（出口参数）

计算过程

  1. 开始；
  2. *pMonth = 1;
  3. 从1到12月，逐月计算
    1. 如果yearDay大于当前月的天数，则，
      1. 将yearDay减去当前月的天数；
      2. *pMonth = 当前的月份;
    2. 否则，
      1. *pDay = yearDay;
      2. 停止计算；
  4. 结束。
